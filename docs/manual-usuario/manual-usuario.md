# Manual de usuario

1. Ingresa con `admin@smartinventory.local` y `Password123`.
2. Revisa el dashboard.
3. Crea categorias y productos.
4. Registra entradas o salidas desde Inventario.
5. Resuelve alertas desde Alertas.
6. Usa IoT > Simulador para generar lecturas.
7. Consulta Analytics para reposicion y baja rotacion.
